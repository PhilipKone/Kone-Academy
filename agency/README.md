# Kone Academy AI Agency

Welcome to the internal AI Agency of Kone Academy. This directory contains a structured collection of AI expert personas designed to deliver premium, world-class results across all divisions.

## Directory Structure

- `core/`: The upstream `agency-agents` repository. DO NOT modify files in this directory.
- `custom/`: Kone Academy's specialized overrides and the **Global Kone DNA**.
- `archive/`: (Optional) Deprecated or old agent versions.

1. **Premium Aesthetics**: Mandatory glassmorphism and elite design.
2. **Engineering Integrity**: Secure, typed, and performant code.
3. **High-Authority Voice**: Professional, visionary, and data-driven communication.
4. **Agentic Continuity**: Leveraging existing Knowledge Items (KIs) and conversation history.

## How to Use These Agents

### For Antigravity (Assistant)
When assigning a task to me, you can specify an agent name (e.g., "Act as the Kone UI Designer"). I will reference the corresponding file in `agency/custom/` or `agency/core/` to adopt that persona's mission, rules, and workflows.

### For Other Tools (Claude Code, Cursor, Aider)
You can point these tools to the `agency/` directory or copy specific `.md` files into their respective agent folders.
