Appendix B: Curriculum Scope and Sequence Documents

B.1 Level 1 - Coding Fundamentals Curriculum

B.1.1 Learning Objectives Mapping

Core Computational Thinking Competencies:
- Algorithmic Thinking: Develop step-by-step problem-solving procedures
- Pattern Recognition: Identify and analyze repeating structures
- Decomposition: Break complex problems into manageable components
- Abstraction: Focus on essential information while ignoring irrelevant details
- Debugging: Systematically identify and fix code errors

Ghanaian Curriculum Integration:
- Mathematics: Problem-solving strategies, logical reasoning
- Science: Scientific method, experimental design
- Language Arts: Sequential thinking, clear communication
- Social Studies: Process analysis, cause-effect relationships

B.1.2 Skill Progression Sequence

Module 1: Introduction to Computational Thinking (Weeks 1-2)
- Computational thinking concepts through Ghanaian daily activities
- Introduction to block-based programming interface
- Simple sequencing activities (market shopping routines, school day schedules)
- Pattern identification in cultural patterns (Kente cloth designs, Adinkra symbols)

Module 2: Basic Programming Concepts (Weeks 3-4)
- Movement and positioning commands
- Simple loop structures for repetitive actions
- Event handling (user interaction, timing)
- Cultural projects: Traditional dance sequences, farming routines

Module 3: Problem Decomposition (Weeks 5-6)
- Breaking complex problems into smaller steps
- Sequential programming with multiple steps
- Simple conditional logic (if-then statements)
- Projects: Market calculation systems, weather prediction tools

Module 4: Pattern Recognition and Application (Weeks 7-8)
- Identifying and utilizing patterns in programming
- Loop optimization and efficiency
- Variable introduction and data storage
- Projects: Agricultural pattern analysis, music rhythm programming

Module 5: Debugging and Refinement (Weeks 9-10)
- Systematic error identification strategies
- Code testing and improvement cycles
- Peer review and collaborative debugging
- Projects: Community service applications, educational games

Module 6: Creative Application (Weeks 11-12)
- Open-ended project development
- Integration of all learned concepts
- Presentation and sharing of completed projects
- Community showcase and celebration

B.1.3 Activity Breakdown by Competency

Algorithmic Thinking Activities:
- Daily routine programming (morning preparation, school activities)
- Recipe following and adaptation (Ghanaian dish preparation)
- Navigation and direction programming (community mapping)
- Story sequencing with logical flow

Pattern Recognition Activities:
- Cultural pattern analysis (Kente cloth mathematical patterns)
- Natural pattern identification (seasonal changes, market cycles)
- Musical pattern programming (traditional Ghanaian rhythms)
- Mathematical pattern extension (number sequences, geometric patterns)

Decomposition Activities:
- Complex problem breaking (community challenges, environmental issues)
- Project planning and organization (school event planning)
- Multi-step procedure design (farming process optimization)
- System analysis (water collection and purification)

Abstraction Activities:
- Essential information identification (problem simplification)
- Generalization from specific examples (pattern extraction)
- Model creation (simplified representations of complex systems)
- Conceptual mapping (relationship visualization)

Debugging Activities:
- Error pattern recognition and categorization
- Systematic testing procedures
- Collaborative problem-solving approaches
- Solution documentation and sharing
