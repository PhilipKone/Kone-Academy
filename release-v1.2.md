# v1.2 - Aesthetic Overhaul & Dynamic UI

## ✨ New Features
- **Neon Theme Engine**: Added a sleek, glassmorphic dropdown to dynamically switch between 5 custom themes: Neon Blue, Matrix Green, Cyberpunk Pink, Lava Orange, and Galactic Purple.
- **Dynamic Brand Logos**: The Header, Footer, and Browser Tab Favicon now perfectly sync with your chosen active theme across the entire application without page reloads.
- **Upgraded App Launcher**: The ecosystem dropdown menu now features Apple-like smooth staggered pop-in animations, a streamlined single-column layout, and clear descriptive text for each platform (Code, Lab, Consult).

## 🎨 Design Enhancements
- **Crisp SVG Logos**: Removed all subtle droplet watermarks/shadows from the SVG files (including the `app-code.svg` icon) for a flawlessly solid, modern, flat-design look.
- **Career Section Standardization**: Ensured all headers and section titles align beautifully with the global aesthetic.

## 🛠️ Optimizations
- Implemented static rendering build fixes (`patch-react-snap.cjs`) to ensure lightning-fast SEO, PWA readiness, and crawler performance.
